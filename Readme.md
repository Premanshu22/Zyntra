# 🎮 ZYNTRA – The Connected Gaming Universe

A cinematic, scroll-driven gaming experience built with React, GSAP, and TailwindCSS.

---

## 🌐 Live Demo

🔗 https://zyntra-seven.vercel.app/

---

## 🚀 About The Project

ZYNTRA is a futuristic gaming ecosystem concept that blends immersive storytelling, motion design, and interactive UI to create a high-impact web experience.

The project focuses on:

- Advanced scroll-triggered animations
- 3D tilt and geometric transitions
- Cinematic hero sections
- Modular component architecture
- Responsive performance-first design

---

## 🛠 Tech Stack

- React (Vite)
- GSAP
- TailwindCSS
- React Icons
- Custom Font Integration

---

## ✨ Core Features

- 🔥 Scroll-based animation engine  
- 🎬 Clip-path geometric transitions  
- 🧊 3D hover tilt effects  
- 🎮 Interactive Bento Grid layout  
- 🌌 AI companion module  
- 📱 Fully responsive across devices
